<p>CURL is AWESOME to do what you want ! It's a simple but effective command line tool : <a href="https://curl.haxx.se/" rel="noreferrer">https://curl.haxx.se/</a></p>

<p>Rest implementation test commands : </p>

<pre><code>curl -i -X GET http://rest-api.io/items
curl -i -X GET http://rest-api.io/items/5069b47aa892630aae059584
curl -i -X DELETE http://rest-api.io/items/5069b47aa892630aae059584
curl -i -X POST -H 'Content-Type: application/json' -d '{"name": "New item", "year": "2009"}' http://rest-api.io/items
curl -i -X PUT -H 'Content-Type: application/json' -d '{"name": "Updated item", "year": "2010"}' http://rest-api.io/items/5069b47aa892630aae059584
</code></pre>

<p>Forget browser and try CLI. <a href="https://httpie.org/" rel="nofollow noreferrer">HTTPie</a> is great tool!</p>

<p><a href="https://i.stack.imgur.com/1VjEt.png" rel="nofollow noreferrer"><img src="https://i.stack.imgur.com/1VjEt.png" alt="enter image description here"></a></p>

<p><strong>CLI</strong> http clients:</p>

<ul>
<li><a href="https://httpie.org/" rel="nofollow noreferrer">HTTPie</a></li>
<li><a href="http://http-prompt.com/" rel="nofollow noreferrer">HTTP Prompt</a></li>
<li><a href="https://curl.haxx.se/docs/httpscripting.html" rel="nofollow noreferrer">Curl</a></li>
<li>wget</li>
</ul>

<p>If you insist on browser extension then:</p>

<p><strong>Chrome</strong>:</p>

<ul>
<li><a href="https://chrome.google.com/webstore/detail/postman/fhbjgbiflinjbdggehcddcbncdddomop" rel="nofollow noreferrer">Postman - REST Client</a> (best browser client)</li>
<li><a href="https://chrome.google.com/webstore/detail/advanced-rest-client/hgmloofddffdnphfgcellkdfbfbjeloo" rel="nofollow noreferrer">Advanced REST client</a></li>
</ul>

<p><strong>Firefox</strong>:</p>

<ul>
<li><a href="https://addons.mozilla.org/en-US/firefox/addon/rest-easy/" rel="nofollow noreferrer">REST Easy</a> (has nice design)</li>
<li><a href="https://addons.mozilla.org/en-US/firefox/addon/restclient/" rel="nofollow noreferrer">RESTClient</a></li>
<li><a href="https://addons.mozilla.org/en-US/firefox/addon/poster/" rel="nofollow noreferrer">Poster plugin</a></li>
</ul>

<p><strong>Firefox</strong></p>

<p>Open Network panel in Developer Tools by pressing <kbd>Ctrl-Shift-Q</kbd> or by going Firefox -> Developer Tools -> Network Tab (highlighted). Then Click on small door icon on top-right (in expanded form in the screenshot, you'll find it just left of the highlighted Headers), second row (if you don't see it then reload the page) -> Edit and resend whatever request you want</p>

<td class="answercell">

<p>Having been greatly inspired by <a href="https://chrome.google.com/webstore/detail/postman/fhbjgbiflinjbdggehcddcbncdddomop">Postman for Chrome</a>, I decided to write something similar for Firefox.</p>

<p><a href="https://addons.mozilla.org/en-US/firefox/addon/rest-easy/">REST Easy</a>* is a restartless Firefox add-on that aims to provide as much control as possible over requests. The add-on is still in an experimental state <s>(it hasn't even been reviewed by Mozilla yet)</s> but development is progressing nicely.</p>

<p><img src="https://addons.cdn.mozilla.net/user-media/previews/full/161/161956.png?modified=1437785730" alt=""></p>

<p>The project is open source, so if anyone feels compelled to help with development, that would be awesome: <a href="https://github.com/nathan-osman/Rest-Easy">https://github.com/nathan-osman/Rest-Easy</a></p>

<p><sub>* the add-on available from <a href="http://addons.mozilla.org">http://addons.mozilla.org</a> will always be slightly behind the code available on GitHub</sub></p>

<p>You specifically asked for "extension or functionality in Chrome and/or Firefox", which the answers you have already received provide, but I do like the simplicity of <a href="https://stackoverflow.com/a/3307401/575530">oezi's answer to the closed question "how to send a post request with a web browser"</a> for simple parameters. oezi says:</p>

<p>with a form, just set <code>method</code> to <code>"post"</code></p>

<pre><code>&lt;form action="blah.php" method="post"&gt;
  &lt;input type="text" name="data" value="mydata" /&gt;
  &lt;input type="submit" /&gt;
&lt;/form&gt;
</code></pre>

<p>I.e. build yourself a very simple page to test the post actions.</p>
